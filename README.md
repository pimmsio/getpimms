## Tech Stack

- **Framework:** Next.js (TypeScript)  
- **Styles:** Tailwind CSS  
- **Database:** PlanetScale  
- **Authentication:** NextAuth.js (+ SAML/SSO via BoxyHQ)  
- **Payments:** Stripe  
- **Emails:** Resend  
- **Monorepo:** Turborepo  
- **Deployments:** Vercel  
- **Link Scanning:** Pangea
